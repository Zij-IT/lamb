# The Lamb Lang

## What is Lamb?

Lamb is a small strongly typed language, which looks like Haskell, Rust and Lua had a baby.
Like in Haskell, all items are immutable, including data-structures like lists and hashmaps.
Like in Rust, functions are called with `()`, and there exists the usual suspects of an imperative
language `for`, `while` and `loop`.

Additionally, Lamb is designed to be able to be loaded into Rust programs similar to how easily Lua
embeds into C and C++.

## Why 'Lamb'

The name was chosen after I decided what kind of language I wanted to write. I knew I wanted to
write a functional language. Many functional languages are based off of <strong>lamb</strong>da
calculus. Nothing more to it!

## Who is Lamb For

Lamb aims to provide a more functional approach to programming, without jumping so far as to mark
`println` as impure. It was written to be able to provide those looking to learn more functional
concepts an outlet, without requiring they forgo the concepts they know like iteration.
